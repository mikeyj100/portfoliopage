Overstimulation Project Summary

This project focused on developing a mobile app prototype to tackle overstimulation, a social-technological problem. The app aims to help users manage time, reduce digital distractions, and promote mindfulness. It incorporates Pomodoro Technique elements and repurposes BeReal's concept for promoting breaks and mindfulness. The main components of the project included:

1. Mobile App Prototype

Created using Figma, inspired by the Pomodoro Technique and BeReal concept. Features include:

  • 25-minute timer for work/study periods
  • 5-minute breaks with photo-taking encouragement
  • Screen dimming for subsequent work sessions

2. Analysis Report

An analysis of overstimulation, its effects, and how the proposed solution addresses the issue. View Report

3. Design Brief

Outlined project goals, target audience, and key app features. View Design Brief

4. Style Guide

Utilized the Uber UI Toolkit in Figma for a user-friendly experience aligned with established design patterns. View Style Guide
