Digital Transformation Project for Bristol Soup Run Trust

This project aimed to rebrand and digitally transform the Bristol Soup Run Trust, a local charity supporting the homeless, to better engage with the student demographic.

View Dissertation Report

The key deliverables included:

  • A redesigned, modern website with improved user experience
  • A mobile app prototype with gamification elements

The project involved extensive user research, including surveys and interviews with students, to understand their motivations, limitations, and preferences regarding charitable engagement. UX/UI principles were applied to create a user-centric design that resonates with students while maintaining the charity's mission.

Key features of the redesign include:

  • A clean, minimalist design with a dark theme
  • A single-page scrolling website for easy navigation
  • Interactive elements to enhance user engagement
  • A QR code system in the mobile app for event participation and rewards

The project successfully created a modern brand identity for the Bristol Soup Run Trust, potentially increasing student awareness and engagement with the charity's cause.
