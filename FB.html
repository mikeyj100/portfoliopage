First Bus App Analysis Project Summary

This project involved analyzing the First Bus app from the perspective of an older retired person.

View Documentation

The analysis covered several key areas:

  • Overview of First Bus and its services
  • Common problems faced by older retired people
  • Current accessibility features within the First Bus app
  • Visual analysis of the app's interface
  • User journey mapping
  • Design brief highlighting key aspects and challenges

The project identified two main challenges for older users:

  1. Lack of experience with advanced technology
  2. Difficulty in seeing key features or small text due to impaired vision

Based on these findings, design recommendations were proposed, including:

  • A simplified initial interface focusing on the core function of planning a journey
  • A key system to explain time stamps for users with cognitive impairments like dementia

The project aimed to improve the app's usability for older retired persons, addressing their specific needs and challenges in using public transportation services.
